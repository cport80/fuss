Welcome to FUSS v0.1


Fuss is a starter style sheet to get your HTML looking better in under 20 seconds. Just add one line to your HTML...


<link rel="stylesheet" type="text/css" href="fuss.css">


Download FUSS and copy the fuss.css file into your HTML sources folder then add the above line of code into your HTML (between the <head> and </head> tags). FUSS does all the rest for you

FUSS sets some CSS styles with better default values to give a modern mobile feel to the most basic HTML pages. It uses percentage based sizing where possible so layout still looks good no matter what device you are using to view your pages. It requires no additional scripts, no extra frameworks and can be customised if something is not to your liking.

FUSS was born out of a need for a quick bootstrap lookalike starting-point which would still work even when javascript is disabled on the browser.
